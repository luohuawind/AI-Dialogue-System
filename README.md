# AI-Dialogue-System
独属于你的私人AI对话系统（本地部署，断网可用)

## 一句话介绍
在笔记本上运行的本地大模型角色对话系统，支持文字+语音输出

## 技术栈
- 模型：Qwen-14B-Instruct (GGUF量化)
- 推理框架：llama.cpp
- 语音：Coqui TTS（可选）/浏览器语音（默认）
- 前端：HTML/CSS/JS
- 后端：Python



## 功能
1. 本地运行，不需要联网API
2. 支持角色设定，对话风格贴合人设
3. 语音合成输出
4. 适配网页/桌面三端
5. 支持记忆导出，导出后聊天记录一键清空
6.一句话：类似猫箱角色扮演对话，只不过断网可用、完全私有、不用担心限制词等问题。 



## 运行方法

1. 下载Qwen 14b模型文件(或者其他你喜欢的，需要改名称)
2. 使用code里的代码
3. 浏览器打开 `http://localhost:8080`



## 踩过的坑
- 模型加载内存不足：改用GGUF量化版本（选择匹配你电脑设置的模型）
- 语音合成延迟：预加载模型
